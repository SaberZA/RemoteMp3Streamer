RemoteMp3Streamer
=================

Stream Mp3 file from remote server via node.js http server.

Thanks to Paulorossi for code references to streaming: https://gist.github.com/paolorossi/1993068

Mp3Streamer.js is the core application.

Seeking is still unsupported.
Having some trouble getting seeking working, but the seeked position is currently available.
